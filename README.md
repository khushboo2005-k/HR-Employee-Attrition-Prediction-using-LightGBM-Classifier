📌 Project Overview

This project focuses on predicting employee attrition using the LightGBM (Light Gradient Boosting Machine) Classifier on the IBM HR Analytics Employee Attrition dataset. The goal is to build an efficient and accurate machine learning model that predicts whether an employee is likely to leave the organization, helping HR teams make informed, data-driven decisions.

🎯 Objective

Develop a classification model using the LightGBM Classifier to accurately predict employee attrition and provide insights that support employee retention strategies and workforce planning.
